How MelodyBOT works

MelodyBOT aims to help composers/songwriters overcome writer's block. If you have a short MIDI melody, you can use melodyBOT to generate a continuation of it.

MelodyBOT is designed to complete short melodies, make sure your file is not longer than 10 seconds.

It is also possible to generate a fully random new melody. To do this, just choose a model and a length, then press generate.

Models

MelodyBOT uses recurrent neural network(RNN) models provided by the Magenta melody_rnn project.

Basic

This configuration acts as a baseline for melody generation with an LSTM(Long Short-Term Memory) model. As the name suggests, expect basic melody outcomes.

Lookback

This model more easily repeats events from 1 and 2 bars ago. This results in melodies that wander less and have a more musical structure.

Attention

Attention allows the model to more easily access past information without having to store that information in the RNN cell's state. This allows the model to more easily learn longer term dependencies, and results in melodies that have longer arching themes.
